# v0.1.0

Initial public release of `codex-portable-memory`.

## Highlights

- Added `codex-memory-mcp`, a LanceDB-backed MCP memory server with:
  - hybrid vector + BM25 recall
  - project-aware scope resolution
  - memory import/export
  - compaction and supersede support
- Added `workspace-memory`, a Markdown-first logging layer with:
  - workspace daily checkpoints
  - per-project session logs
  - scheduled task registration
- Added `portable-memory`, a reusable Codex skill for:
  - recall-before-work workflow
  - durable memory storage rules
  - portable migration and cleanup flow

## Validation

- `packages/codex-memory-mcp`
  - `npm test`
  - `npm run typecheck`
- `workspace-memory`
  - `Invoke-Pester -Path workspace-memory/tests`
- `skills/portable-memory`
  - validated with Codex skill validator

## Notes

- This repository publishes the framework only.
- Personal memory data, local databases, exports, and real session logs are intentionally excluded.
